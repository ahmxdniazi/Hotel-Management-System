## [1.0.0] — 2025

### Added
- Room management: add, update, delete, and view rooms
- Guest registration and record management
- Booking/reservation system with check-in and check-out
- Invoice and billing generation
- Search functionality for guests and rooms
- MySQL database integration via JDBC
- NetBeans-compatible project structure with Ant build
